Grid

Use class='container' or class='container-fluid' to define a grid layout. Each layout has multiple row where are defined columns:

<div class="container">
  <fieldset class="form-group">
    <legend>Attributes</legend>

    <div class="row">
      <div class="col col-md-4"><!-- content --></div>
      <div class="col col-md-8"><!-- content --></div>
    </div>
  </fieldset>
</div>
